## How to add a field

Simply click or drag and drop your chosen **field type** from the left-hand panel, then type your question or text.

![Screenshot 2025-10-02 at 3.39.58â€¯PM.png](/images/Screenshot2025-10-02at3.39.58PM.png)

Each field is highly customizable. Just click on any field followed by its <Icon icon="gear" iconType="solid" /> to open the right-hand panel.

Add supporting details, make it required, set up [**conditional logic**](), and more. You can also easily [copy and paste](/copy-paste-fields) or [duplicate](/duplicate-fields) your field types.

## Change field type

Click a field followed by <Icon icon="arrow-right-arrow-left" iconType="solid" /> to select a different field type.

<Check>
  Changing your field to the suggested field types will keep your current configurations.
</Check>

## List of field types

| **Field type**                                                                                     | **Purpose**                                                                                                |
| -------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| Short answer                                                                                       | Collect a single line of text response                                                                     |
| Multiple choice                                                                                    | Present a list of choices to select only one option from                                                   |
| Email input                                                                                        | Collect an email address                                                                                   |
| Text                                                                                               | Display a block of text                                                                                    |
| Phone number                                                                                       | Collect a phone number                                                                                     |
| [Address](/address)                                                                                | Collect a mailing address                                                                                  |
| [Heading](/heading)                                                                                | Display a heading in the largest size                                                                      |
| Paragraph                                                                                          | Display a formattable block of text in a paragraph format                                                  |
| [Banner](/banners)                                                                                 | Display a banner image or message                                                                          |
| Dropdown                                                                                           | Present a list of options in a dropdown menu format                                                        |
| [Picture choice](/image-gif#1d80b0099a324fbd8301e152b995d9ea)                                      | Show a set of image options to select from                                                                 |
| Multiselect                                                                                        | Present a list of options to choose from, with the ability to select multiple options                      |
| Switch                                                                                             | Show a toggle switch to turn an option on or off                                                           |
| Checkbox                                                                                           | Show a yes or no question, like a terms of service agreement checkbox                                      |
| Checkboxes                                                                                         | Present a list of options to choose from, with the ability to tick multiple options                        |
| Record picker                                                                                      | Present a list of records to select from (only for forms connected to a database like Airtable and Notion) |
| [Submission picker](/submission-picker)                                                            | Enable pre-filling of forms with responses from another form                                               |
| [Choice matrix](/matrix)                                                                           | Collect a grid of selections from corresponding rows and columns                                           |
| Date picker                                                                                        | Collect a date value                                                                                       |
| Date time picker                                                                                   | Collect a date and time value                                                                              |
| Time picker                                                                                        | Collect a time value                                                                                       |
| [Date range](/date-range)                                                                          | Collect a start and end date                                                                               |
| [Ranking](/ranking)                                                                                | Show a list of options to rank                                                                             |
| [Star Rating](/rating)                                                                             | Show a star rating system to rate an attribute                                                             |
| [Slider](/rating)                                                                                  | Show a sliding control to select a value from a range                                                      |
| [Opinion scale](/rating)                                                                           | Show a predefined scale to express opinion on a topic                                                      |
| Long answer                                                                                        | Collect a longer text response, such as a paragraph or a list                                              |
| Number                                                                                             | Collect a numerical value                                                                                  |
| [Currency](/currency) <Icon icon="gem" iconType="solid" color="#b8d8e7" />                         | Collect a monetary amount                                                                                  |
| [URL](/add-links-to-form)                                                                          | Request for website links                                                                                  |
| Color picker                                                                                       | Present a predefined set of options or using a color picker interface                                      |
| Password                                                                                           | Collect a password                                                                                         |
| [File uploader](/file-upload)                                                                      | Collect file uploads                                                                                       |
| [Signature](/esignatures) <Icon icon="gem" iconType="solid" color="#b8d8e7" />                     | Request for a free-form or upload of a signature                                                           |
| [Voice recording](/voice-recording)                                                                | Request voice recordings and automatically transcribe them                                                 |
| [Subforms](/subforms)                                                                              | Collect information on nested forms within your main form                                                  |
| [Captcha](/captcha) <Icon icon="gem" iconType="solid" color="#b8d8e7" />                           | Present respondents with a challenge to prove they are human and not a bot                                 |
| [Location coordinates](/location-coordinates) <Icon icon="gem" iconType="solid" color="#b8d8e7" /> | Collect geographic location coordinates                                                                    |
| [Section collapse](/section-collapse)                                                              | Allow respondents to show or hide a section of the form                                                    |
| Divider                                                                                            | Display a horizontal line to visually separate content on your form                                        |
| [HTML](/html-css) <Icon icon="gem" iconType="solid" color="#b8d8e7" />                             | Control the layout of your form elements and add custom styles to it                                       |
| [**Image**](/image-gif)                                                                            | Display an image or GIF by uploading one or choosing from a gallery                                        |
| [Video](/video)                                                                                    | Display a video                                                                                            |
| [PDF Viewer](/embed-pdfs) <Icon icon="gem" iconType="solid" color="#b8d8e7" />                     | Display a PDF document                                                                                     |
| [Progress bar](/progress-bar)                                                                      | Show a visual indicator to help respondents track their completion                                         |
| [Fill again button](/fill-again-button)                                                            | Let respondents fill out your form multiple times (only on ending pages)                                   |
| Send response                                                                                      | Let receive a copy of their responses via email (only on ending pages)                                     |
| Download PDF                                                                                       | Let respondents download the PDF copy generated after completing their form (only on ending pages)         |
| [Table](/table)                                                                                    | Collect data in a spreadsheet-like format with multiple rows and columns                                   |
| [Social media links](/social-media)                                                                | Add buttons that link to your social media pages, messaging channels, portfolios, and more                 |
