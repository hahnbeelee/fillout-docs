---
title: Create Record
description: Creates a new record in a table.
api: POST /api/v1/bases/{basePublicId}/tables/{tableId}/records
---

Creates a new record in a table with the provided field data.

## Path Parameters

<ParamField path="basePublicId" type="string" required>
  The unique identifier of the database
</ParamField>

<ParamField path="tableId" type="string" required>
  The unique identifier of the table
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

## Request Body

<ParamField body="record" type="object" required>
  Record data with field names as keys and their corresponding values

  <Expandable title="Record object structure">
    <ParamField body="[fieldName]" type="any">
      Field values should match the field types defined in your table schema
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "record": {
      "Name": "John Doe",
      "Email": "john@example.com",
      "Priority": "high"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    record: {
      "Name": "John Doe",
      "Email": "john@example.com",
      "Priority": "high"
    }
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records',
    headers={
        'Authorization': 'Bearer your_api_key',
        'Content-Type': 'application/json'
    },
    json={
        'record': {
            'Name': 'John Doe',
            'Email': 'john@example.com',
            'Priority': 'high'
        }
    }
)
```

</RequestExample>

## Response

Returns the newly created record with generated ID and timestamps.

<ResponseField name="id" type="string">
  Auto-generated UUID for the record
</ResponseField>

<ResponseField name="data" type="object">
  The record data containing all field values
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO timestamp of when the record was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO timestamp of when the record was last updated
</ResponseField>

<ResponseExample>

```json Response
{
  "id": "rec_uuid123",
  "data": {
    "Name": "John Doe",
    "Email": "john@example.com", 
    "Priority": "high"
  },
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

</ResponseExample>

<Note>
  - Field names should match exactly what you defined in your table schema
  - Record IDs are automatically generated - any provided `id` values are ignored
  - For linked record fields, provide an array of related record IDs
</Note>
