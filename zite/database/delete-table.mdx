---
title: Delete Table
description: Permanently deletes a table and all its records.
api: DELETE /api/v1/bases/{basePublicId}/tables/{tableId}
---

Permanently deletes a table and all its records.

## Path Parameters

<ParamField path="basePublicId" type="string" required>
  The unique identifier of the database
</ParamField>

<ParamField path="tableId" type="string" required>
  The unique identifier of the table to delete
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<RequestExample>

```bash cURL
curl -X DELETE https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789 \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});
```

```python Python
import requests

response = requests.delete(
    'https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789',
    headers={
        'Authorization': 'Bearer your_api_key'
    }
)
```

</RequestExample>

## Response

Returns a 204 No Content response when the table is successfully deleted.

<ResponseExample>

```json Response
204 No Content
```

</ResponseExample>

<Warning>
  Deleting a table will permanently remove the table, all its fields, views, and all records stored in the table. This action cannot be undone.
</Warning>
