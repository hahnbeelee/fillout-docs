---
title: Get Record by ID
description: Retrieves a specific record by UUID.
api: GET /api/v1/bases/{databaseId}/tables/{tableId}/records/{recordId}
---

Retrieves a specific record by its unique identifier.

## Path Parameters

<ParamField path="databaseId" type="string" required>
  The unique identifier of the database
</ParamField>

<ParamField path="tableId" type="string" required>
  The unique identifier of the table
</ParamField>

<ParamField path="recordId" type="string" required>
  The UUID of the record to retrieve
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Enter your Fillout API key. Format: Bearer &lt;api_key&gt;
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records/rec_uuid123 \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records/rec_uuid123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});
```

```python Python
import requests

response = requests.get(
    'https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records/rec_uuid123',
    headers={
        'Authorization': 'Bearer your_api_key'
    }
)
```

</RequestExample>

## Response

Returns the requested record with all field data.

<ResponseField name="id" type="string">
  The unique UUID of the record
</ResponseField>

<ResponseField name="data" type="object">
  The record data containing all field values
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO timestamp of when the record was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO timestamp of when the record was last updated
</ResponseField>

<ResponseExample>

```json Response
{
  "id": "rec_uuid123",
  "data": {
    "Name": "John Doe",
    "Email": "john@example.com",
    "Priority": "high"
  },
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

</ResponseExample>

<Warning>
  Record ID must be in valid UUID format. Invalid formats will return a 400 error.
</Warning>
