---
title: Create Table
description: Adds a new table to an existing database.
api: POST /api/v1/bases/{databaseId}/tables
---

Adds a new table to an existing database.

## Path Parameters

<ParamField path="databaseId" type="string" required>
  The unique identifier of the database
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Enter your Fillout API key. Format: Bearer &lt;api_key&gt;
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Table name (minimum 1 character)
</ParamField>

<ParamField body="fields" type="array" required>
  Array of field definitions to create with the table (at least one field required)

  <Expandable title="Field definition structure">
    <ParamField body="type" type="string" required>
      Field type (see [Field Types Reference](/zite/database/field-types))
    </ParamField>
    <ParamField body="name" type="string" required>
      Field name (minimum 1 character)
    </ParamField>
    <ParamField body="template" type="object" required>
      Field-specific configuration options that vary by field type
    </ParamField>
  </Expandable>
</ParamField>

## Response

Returns the newly created table with generated ID, field definitions, and views.

<ResponseField name="id" type="string">
  Auto-generated unique identifier for the table (starts with `tbl_`)
</ResponseField>

<ResponseField name="name" type="string">
  The table name that was specified in the request
</ResponseField>

<ResponseField name="order" type="number">
  The display order of the table within the database
</ResponseField>

<ResponseField name="primaryFieldId" type="string">
  The ID of the first field, which serves as the primary field
</ResponseField>

<ResponseField name="fields" type="array">
  Array of created field objects with their generated IDs and configurations
</ResponseField>

<ResponseField name="views" type="array">
  Array of default views created for the table (typically includes a grid view)
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO timestamp of when the table was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO timestamp of when the table was last updated
</ResponseField>
