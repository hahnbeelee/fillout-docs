---
title: Delete Database
description: Permanently deletes a database and all its data.
api: DELETE /api/v1/bases/{databaseId}
---

Permanently deletes a database and all its data.

## Path Parameters

<ParamField path="databaseId" type="string" required>
  The unique identifier of the database to delete
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Enter your Fillout API key. Format: Bearer &lt;api_key&gt;
</ParamField>

<RequestExample>

```bash cURL
curl -X DELETE https://api.fillout.com/api/v1/bases/base_abc123 \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/api/v1/bases/base_abc123', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});
```

```python Python
import requests

response = requests.delete(
    'https://api.fillout.com/api/v1/bases/base_abc123',
    headers={
        'Authorization': 'Bearer your_api_key'
    }
)
```

</RequestExample>

## Response

Returns a 204 No Content response when the database is successfully deleted.

<ResponseExample>

```json Response
204 No Content
```

</ResponseExample>

<Warning>
  Deleting a database will permanently remove the entire database including all tables, fields, views, and records. This action cannot be undone.
</Warning>
