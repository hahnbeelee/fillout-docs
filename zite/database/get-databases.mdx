---
title: Get Databases
description: Lists all databases for your organization.
api: GET /api/v1/bases
---

Lists all databases for your organization.

## Headers

<ParamField header="Authorization" type="string" required>
  Enter your Fillout API key. Format: Bearer &lt;api_key&gt;
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.fillout.com/api/v1/bases \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/api/v1/bases', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});
```

```python Python
import requests

response = requests.get(
    'https://api.fillout.com/api/v1/bases',
    headers={
        'Authorization': 'Bearer your_api_key'
    }
)
```

</RequestExample>

## Response

Returns an array of all databases accessible to your organization.

<ResponseField name="bases" type="array">
  Array of database objects
  
  <Expandable title="Database object structure">
    <ResponseField name="id" type="string">
      The unique identifier of the database
    </ResponseField>
    <ResponseField name="name" type="string">
      The database name
    </ResponseField>
    <ResponseField name="workspaceId" type="number">
      The workspace ID where the database belongs
    </ResponseField>
    <ResponseField name="tables" type="array">
      Array of table objects with their complete structure including fields and views
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      ISO timestamp of when the database was created
    </ResponseField>
    <ResponseField name="updatedAt" type="string">
      ISO timestamp of when the database was last updated
    </ResponseField>
  </Expandable>
</ResponseField>
