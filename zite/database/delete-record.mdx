---
title: Delete Record
description: Permanently removes a record from a table.
api: DELETE /api/v1/bases/{basePublicId}/tables/{tableId}/records/{recordId}
---

Permanently removes a record from a table.

## Path Parameters

<ParamField path="basePublicId" type="string" required>
  The unique identifier of the database
</ParamField>

<ParamField path="tableId" type="string" required>
  The unique identifier of the table
</ParamField>

<ParamField path="recordId" type="string" required>
  The UUID of the record to delete
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<RequestExample>

```bash cURL
curl -X DELETE https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records/rec_uuid123 \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records/rec_uuid123', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});
```

```python Python
import requests

response = requests.delete(
    'https://api.fillout.com/api/v1/bases/base_abc123/tables/tbl_xyz789/records/rec_uuid123',
    headers={
        'Authorization': 'Bearer your_api_key'
    }
)
```

</RequestExample>

## Response

Returns a 204 No Content response when the record is successfully deleted.

<ResponseExample>

```json Response
204 No Content
```

</ResponseExample>

<Warning>
  Record ID must be in valid UUID format. Invalid formats will return a 400 error.
</Warning>
