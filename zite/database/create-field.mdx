---
title: Create Field
description: Adds a new field to an existing table.
api: POST /api/v1/bases/{databaseId}/tables/{tableId}/fields
---

Adds a new field to an existing table.

## Path Parameters

<ParamField path="databaseId" type="string" required>
  The unique identifier of the database
</ParamField>

<ParamField path="tableId" type="string" required>
  The unique identifier of the table
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Enter your Fillout API key. Format: Bearer &lt;api_key&gt;
</ParamField>

## Request Body

<ParamField body="type" type="string" required>
  Field type (see [Field Types Reference](/zite/database/field-types))
</ParamField>

<ParamField body="name" type="string" required>
  Field name (minimum 1 character)
</ParamField>

<ParamField body="template" type="object" required>
  Field-specific configuration options that vary by field type

  <Expandable title="Template structure">
    The template object structure depends on the field type. For example:
    - `single_select`: requires `options` array with `value`, `label`, and `color`
    - `number`: requires `format` and optional `precision`
    - `linked_record`: requires `tableId` for the target table
    
    See the [Field Types Reference](/zite/database/field-types) for complete template structures.
  </Expandable>
</ParamField>

## Response

Returns the created field with its generated ID and configuration.

<ResponseField name="id" type="string">
  Auto-generated unique identifier for the field (starts with `fld_`)
</ResponseField>

<ResponseField name="type" type="string">
  The field type that was specified in the request
</ResponseField>

<ResponseField name="name" type="string">
  The field name that was specified in the request
</ResponseField>

<ResponseField name="template" type="object">
  The field configuration options that were specified in the request
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO timestamp of when the field was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO timestamp of when the field was last updated
</ResponseField>
