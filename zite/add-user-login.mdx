## What is user login?

User login controls who can access your app. Zite handles authentication for you and connects to your existing user database (Airtable, Google Sheets, Zite Database, etc.).

## Step by step guide

<Steps>
  <Step title="Add user login">
    Type **"Add user login"** in your Zite chat, or click the **\+** icon and select **Add login**.

    <Frame>
      ![Zite chat with 'Add user login' prompt](/images/auth-step-1.png)
    </Frame>
  </Step>
  <Step title="Choose your database">
    Select where to store user data:

    - **Zite Database**
    - **Airtable**
    - **Google Sheets and other databases (coming soon)**

    Then specify which table and email field to use for login.
  </Step>
  <Step title="Configure access">
    In the **Login** tab, set:

    - **Login methods**: Magic link and/or Google
    - **Who can sign up**: Anyone, invite-only, or specific domains

    <Frame>
      ![Zite access control settings](/images/auth-step-5.png)
    </Frame>
    <Tip>
      While editing your app, you can choose the user you want to log in as from a dropdown populated by the database linked to the login feature. However, the published app requires users to type their email address (or use Google) to log in. This list of users will not appear in the published app.
    </Tip>
  </Step>
  <Step title="Customize login page">
    Click **Edit** to add your logo, colors, and branding.

    <Frame>
      ![Zite login page customization](/images/auth-step-10.png)
    </Frame>
  </Step>
  <Step title="Manage users">
    Use the **Users** tab to view user sessions. To add, edit, or remove users, make changes in your database (e.g. Zite Database or Airtable) directly.

    <Frame>
      ![Zite user management tab](/images/auth-step-4.png)
    </Frame>
  </Step>
</Steps>

## Key features

**User-specific data**: In each workflow, you'll have access to the logged-in user's records automatically.

**Automatic security**: Backend workflows are secured by default. Only logged-in users can call workflows.

**No user limits**: Add unlimited users without per-seat pricing.

**Database integration**: Connect to existing Airtable, Google Sheets, or other databases without duplicating data.
