## Getting Started

<AccordionGroup>
  <Accordion title="What is Zite?">
    [Zite](https://zite.com) is an AI-powered app builder that lets you create web apps, websites, internal tools and portals without writing code.
  </Accordion>
  <Accordion title="How do I create a new Zite?">
    Start at [zite.com](https://zite.com) and type your idea into the prompt. See our [quick start guide](/zite/getting-started) for more details.
  </Accordion>
  <Accordion title="Do I need coding experience?">
    No. Zite is built for both technical and non-technical users. You can rely on AI, use the visual editor, or dive into the code if you prefer.

## Building with Zite

<AccordionGroup>
  <Accordion title="What can I build?">
    Zite is most commonly used to build internal tools, dashboards, portals and business web applications.
  </Accordion>
  <Accordion title="Tips for effective prompts">
    - Be specific about what you want Zite to change
    - Build in small steps
    - Use clear instructions (e.g. "Change the submission button color to blue")
  </Accordion>
  <Accordion title="How do I handle errors?">
    - Use the built-in "fix it for me" option if an error occurs
    - Re-phrase your request with more detail
    - Restore to a previous working version if needed
  </Accordion>
  <Accordion title="Can I connect a database?">
    Yes. Zite has a built-in database or you can connect to external databases like Airtable or Google Sheets.
  </Accordion>

## Troubleshooting

<AccordionGroup>
  <Accordion title="My app isn't loading correctly">
    Try these steps:

    - Ask Zite to fix the issue
    - Restore to a previous working version
    - Contact [support@fillout.com](mailto:support@fillout.com) if the issue persists
  </Accordion>
  <Accordion title="My custom domain isn't setup correctly">
    Verify that:

    - Your DNS records are correctly configured
    - SSL certificate has been generated (this can take up to 24 hours)
    - The domain is properly connected in your project settings

    Contact support at [support@fillout.com](mailto:support@fillout.com) if the issue persists.
  </Accordion>
  <Accordion title="How do I report a bug or submit feedback?">
    Submit feedback and feature requests [here](https://main.fillout.com/zite-feedback). For urgent issues, contact support at [support@fillout.com](mailto:support@fillout.com).
  </Accordion>
  <Accordion title="Where can I get more help?">
    - Check our [help center](/zite/overview)
    - Contact support at [support@fillout.com](mailto:support@fillout.com)
    - Submit feedback [here](https://main.fillout.com/zite-feedback)
  </Accordion>
</AccordionGroup>
