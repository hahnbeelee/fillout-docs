## Overview

Build mode is where Zite **makes changes** to your app. This is the default mode when youâ€™re building features, fixing issues, or updating design.

**When to use Build mode**

- Add or change features (new pages, components, workflows, etc.)
- Fix bugs or unexpected behavior
- Update formatting and styling (layout, colors, text, spacing)
- Set up integrations (databases, APIs, automations)
- Apply a plan you created in Plan mode
- Make any other changes to your app
