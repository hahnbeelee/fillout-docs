Adds a new field to an existing table using either the table ID or table name.

<Note>
Both `type` and `name` are required. The `template` object structure varies by field type - see [Field Types Reference](/database/field-types) for complete details
</Note>

## Example Field Creation

Here's how to create a single select field:

```json
{
  "type": "single_select",
  "name": "Status",
  "template": {
    "options": [
      {"label": "Active", "color": "#10b981"},
      {"label": "Inactive", "color": "#6b7280"},
      {"label": "Pending", "color": "#f59e0b"},
      {"label": "Archived", "color": "#ef4444"}
    ]
  }
}
```
