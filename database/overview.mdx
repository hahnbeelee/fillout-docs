## What's included

- **20+ field types** - Text, numbers, dates, files, linked records, and more
- **CSV import/export** - Bring in existing data or export for analysis
- **REST API + Webhooks** - Programmatic access with 50+ requests/second
- **Linked records** - Create relationships between tables
- **Bulk operations** - Update or delete multiple records at once
- **Granular permissions** - Control who can view or edit your data
- **Undo/redo** - Easily revert changes
- **Multiple views** - Organize and filter your data

## Performance

Zite Database is built for speed:

- **50+ requests per second** - Handle high traffic without slowdowns
- **Millions of records** - Scale your database as your app grows
- **Fast queries** - Get results instantly, even with complex filters

## Managing your database

Access the full database interface:

- Add or remove fields
- Create new tables
- Import data from CSVs
- Bulk edit records
- Set up webhooks
- Use the REST API
