### Zite Database in 2 minutes

Zite includes a built-in database for storing and managing your app data. Create tables, add fields, and manage records without writing SQL or code.

The database integrates directly with your Zite apps and supports over 20 field types including text, numbers, dates, files, and relationships.

## What you can do

- **Create databases and tables** - Organize your data with custom databases and tables
- **Add and manage fields** - Use 20+ field types including text, numbers, dates, files, and relationships  
- **Store and query records** - Add, update, and retrieve data through the interface or API
- **Import existing data** - Bring in data from CSVs, Google Sheets, and Airtable
- **API access** - Programmatically manage your data with the REST API
